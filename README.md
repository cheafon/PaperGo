# PaperGo
A full-stack  agent web client helps researcher get the trend and info about the paper and more than that.

## Feature
1. chat，对各个问题都能智能解析。
   痛点：
    难点：上下文管理，错误的处理，问题覆盖的
2. 知识网络树，形成一个线性故事，从某一年开始，XX作者做出了贡献，转折点（引用网络），现状是如何，难点如何，不要幻觉的0样本生成，利用这个网络去学习。（不知道这个领域的背景，学习有阻碍）
3. 图表展示，这一点是相对简单，是通过api接口去调用，前台用echart对接即可
4. 生成论文
